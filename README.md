# InfectionDeathPenalty

### Description:

Inflicts players with infection when they go down.

A simple mod that adds a layer of difficulty to challenge players to not get incapacitated so often during rundowns. The game is already hard, but this mod is here to make it worse! 

The infection value can be configured in the settings with live edit. Keep in mind that infection can't go over the `85%` limit. The game will just drop any infection back down, to prevent players from constantly dying.

This infection can be resisted with boosters or cleansed with any disinfection method.
